# UniteUs ETL - Documentation

Welcome to the UniteUs ETL documentation. This guide will help you deploy, configure, and troubleshoot the application.

---

## Quick Start

1. **Launch the server:** Double-click `launch.pyw`
2. **Get the URL:** Check the Server Control window for network access URLs
3. **Share with users:** Copy the Network IP URL and share with users on the same network

For detailed setup, see the guides below.

## Getting Help

1. **Check the relevant guide** - Each document covers specific topics
2. **Check server logs** - Located in `data/logs/`
3. **Review Server Control window** - Shows current status and URLs
4. **Check Windows Firewall** - Ensure the port is allowed
